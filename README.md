# Pendataan Pemilih dan Perhitungan Suara Pemilihan Ketua RT

## Deskripsi Proyek
Proyek ini merupakan dokumentasi administrasi pemilihan Ketua RT yang mencakup
pendataan pemilih, rekapitulasi surat suara, dan perhitungan hasil perolehan suara.

Proyek ini didasarkan pada pengalaman nyata dalam kegiatan administrasi pemilihan,
dengan seluruh data dan identitas telah disamarkan untuk menjaga kerahasiaan
dan privasi pihak terkait.

## Ruang Lingkup Pekerjaan
- Penyusunan database pemilih berbasis Excel
- Rekap data pemilih aktif dan tidak aktif
- Pencatatan penggunaan surat suara
- Rekapitulasi perolehan suara calon
- Penyusunan laporan administrasi pemilihan

## Tools yang Digunakan
- Microsoft Excel
- Microsoft Word / PDF

## Output Proyek
- Database_Pemilih.xlsx   
- Laporan_Pemilihan_Ketua_RT.pdf  

## Catatan
Dokumen ini digunakan sebagai portofolio administrasi.
Data yang digunakan bukan data asli dan telah disesuaikan untuk keperluan dokumentasi.
